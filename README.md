# Location Library

> 一个图书管理系统

## 实现技术

- 采用`express/node`开发后台

- 使用`pug`模板引擎编写页面

- 使用`MongDB`数据库

## 功能介绍

### 1. 作者管理

- 添加作者

- 删除作者

- 修改作者

- 查看作者

### 2. 书籍管理

- 添加书籍

- 删除书籍

- 修改书籍

- 查看书籍

### 3. 书籍种类管理

- 添加种类

- 删除种类

- 修改种类

- 查看种类

### 4. 书籍实例信息管理

- 添加书籍实例

- 删除书籍实例

- 修改书籍实例

- 查看书籍种类
